# ACCA Rules v2.1
This document defines the ACCA fallback ladder and tier diversity rules.

## Fallback Ladder
1. Elite (Tier 1, 75%+)
2. Primary Fallback (Tier 1 & 2, Double Chance allowed, 75%+)
3. Strong (Tier 1 & 2, 55%+)
4. Safe (Tier 1, 2, 3, safe Tier 3 only)
5. Fallback (Tier 1-4)

## Tier Diversity
- Max 1 leg per tier per match.
- Tier 1: 1X2
- Tier 2: Double Chance, Over/Under, BTTS
- Tier 3: Exotics
